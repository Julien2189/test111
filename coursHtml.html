01
HTMLStructure de base

Un fichier HTML est un document texte que le navigateur interprète pour afficher une page. Chaque page commence par une structure minimale obligatoire.

HTML
<!-- Déclaration du type de document -->
<!DOCTYPE html>

<!-- Racine de la page, langue française -->
<html lang="fr">

  <head>
    <!-- Encodage des caractères -->
    <meta charset="UTF-8" />

    <!-- Titre visible dans l'onglet -->
    <title>Ma première page</title>

    <!-- Lien vers la feuille de style -->
    <link rel="stylesheet" href="style.css" />
  </head>

  <body>
    <!-- Tout ce qui s'affiche à l'écran va ici -->
    <h1>Bonjour le monde !</h1>
  </body>

</html>
💡 À retenir
<head> contient les métadonnées (invisible à l'écran). <body> contient tout le contenu visible.

Les balises HTML structurent le contenu. Elles s'ouvrent avec <balise> et se ferment avec </balise>.

BaliseRôleExemple

<em>Texte en italique<em>Accent</em>
<ul> / <li>Liste à puces<ul><li>Item</li></ul>
<ol> / <li>Liste numérotée<ol><li>1er</li></ol>
<div>Conteneur générique (bloc)<div>…</div>
<span>Conteneur inline<span>mot</span>

Exemple complet

  <h2>Le HTML en pratique</h2>

  <ul>
    <li>Facile à apprendre</li>
    <li>Lisible par les navigateurs</li>
    <li>Base du développement web</li>
  </ul>

  <img src="logo.png" alt="Logo du site" width="200" />

03
HTMLAttributs & Liens

Les attributs enrichissent les balises : ils apportent des informations supplémentaires. Ils se placent à l'intérieur de la balise ouvrante.

Attributs
<!-- Syntaxe : attribut="valeur" -->
<img src="photo.jpg"
     alt="Description pour l'accessibilité"
     width="400"
     height="300" />

<!-- Lien interne -->
<a href="contact.html">Contactez-moi</a>

<!-- Lien externe, s'ouvre dans un nouvel onglet -->
<a href="https://google.com"
   target="_blank"
   rel="noopener noreferrer">
  Aller sur Google
</a>

<!-- Ancre interne (scroll vers un id) -->
<a href="#section-contact">Aller au contact</a>
<section id="section-contact">…</section>
ℹ️ Sécurité
Toujours ajouter rel="noopener noreferrer" sur les liens target="_blank" pour éviter les failles de sécurité.
04
HTMLFormulaires

Les formulaires permettent de collecter des données auprès de l'utilisateur. La balise <form> enveloppe tous les champs.

Formulaire de contact
<form action="/envoyer" method="post">

  <!-- Champ texte -->
  <label for="nom">Votre nom</label>
  <input type="text" id="nom" name="nom"
         placeholder="Jean Dupont" required />

  <!-- Email -->
  <label for="email">Email</label>
  <input type="email" id="email" name="email" />

  <!-- Zone de texte -->
  <label for="msg">Message</label>
  <textarea id="msg" name="msg" rows="4"></textarea>

  <!-- Liste déroulante -->
  <select name="sujet">
    <option value="info">Information</option>
    <option value="devis">Devis</option>
  </select>

  <!-- Bouton d'envoi -->
  <button type="submit">Envoyer</button>

</form>

05
CSSSélecteurs

Le CSS permet de styliser les éléments HTML. Un fichier CSS est une suite de règles : chaque règle contient un sélecteur et des déclarations.

CSS — Sélecteurs
/* Sélecteur de balise — cible tous les <p> */
p {
  color: #333;
  font-size: 16px;
}

/* Sélecteur de classe — préfixé par un point */
.carte {
  background: white;
  border-radius: 8px;
  padding: 20px;
}

/* Sélecteur d'id — préfixé par un dièse, unique */
#entete {
  background: #1a1a2e;
  color: white;
}

/* Sélecteur combiné : enfant direct */
nav > a {
  text-decoration: none;
}

/* Pseudo-classe : au survol */
button:hover {
  background: #gold;
  cursor: pointer;
}

/* Pseudo-élément : première lettre */
p::first-letter {
  font-size: 2em;
  color: #c9a84c;
}
06
CSSLe Box Model

Chaque élément HTML est représenté comme une boîte rectangulaire. Ce modèle définit comment l'espace est calculé autour du contenu.

MARGIN
BORDER
PADDING
CONTENT
CSS — Box Model
.boite {
  /* Largeur et hauteur du contenu */
  width: 300px;
  height: 150px;

  /* Espace intérieur (entre contenu et bordure) */
  padding: 20px;           /* tous les côtés */
  padding: 10px 20px;      /* haut/bas  gauche/droite */
  padding-top: 10px;       /* côté spécifique */

  /* Bordure */
  border: 2px solid #ccc;
  border-radius: 8px;      /* coins arrondis */

  /* Espace extérieur (entre la boîte et les voisins) */
  margin: 16px auto;      /* centré horizontalement */

  /* Important : inclut padding+border dans la largeur */
  box-sizing: border-box;
}
✅ Bonne pratique
Toujours ajouter *, *::before, *::after { box-sizing: border-box; } en début de CSS. Cela rend le calcul des tailles beaucoup plus intuitif.
07
CSSFlexbox

Flexbox est le système de mise en page le plus utilisé. Il permet d'aligner et distribuer des éléments horizontalement ou verticalement, facilement.

CSS — Flexbox
.conteneur {
  display: flex;                /* Active Flexbox */

  /* Direction des éléments */
  flex-direction: row;          /* row | column | row-reverse */

  /* Alignement sur l'axe principal (horizontal si row) */
  justify-content: space-between; /* flex-start | center | flex-end */

  /* Alignement sur l'axe secondaire (vertical si row) */
  align-items: center;          /* stretch | flex-start | flex-end */

  /* Retour à la ligne automatique */
  flex-wrap: wrap;

  /* Espacement entre les enfants */
  gap: 16px;
}

.enfant {
  /* Proportion : cet enfant prend 2x plus de place */
  flex: 2;

  /* Taille minimale avant que flex joue */
  flex-basis: 200px;
}

08
CSSTypographie

Une bonne typographie rend une page lisible et élégante. Le CSS offre de nombreuses propriétés pour contrôler le texte.

CSS — Typographie
/* Import d'une Google Font */
@import url('https://fonts.googleapis.com/css2?family=Playfair+Display:wght@700&display=swap');

body {
  font-family: 'Playfair Display', serif;
  font-size: 16px;       /* taille de base */
  line-height: 1.6;       /* interligne = 1.6 × font-size */
  color: #222;
}

h1 {
  font-size: clamp(32px, 5vw, 64px); /* responsive */
  font-weight: 900;
  letter-spacing: -0.02em;  /* tracking négatif = serré */
  text-transform: uppercase;
}

p {
  max-width: 65ch;          /* ~65 caractères par ligne */
  text-align: justify;
}

/* Dégradé sur du texte */
.titre-gold {
  background: linear-gradient(135deg, #e8c97a, #c9a84c);
  -webkit-background-clip: text;
  -webkit-text-fill-color: transparent;
}
09
CSSAnimations & Transitions

Les animations CSS donnent vie à une interface sans une seule ligne de JavaScript. On distingue les transitions (changement d'état) et les animations (boucles).

CSS — Transitions
.bouton {
  background: #1a1a2e;
  color: white;
  padding: 12px 28px;
  border-radius: 6px;
  border: none;
  cursor: pointer;

  /* Transition : propriété durée timing */
  transition: background 0.3s ease, transform 0.2s ease;
}

.bouton:hover {
  background: #c9a84c;
  transform: translateY(-2px); /* léger décalage vers le haut */
}
CSS — @keyframes
/* Définir l'animation */
@keyframes apparition {
  from {
    opacity: 0;
    transform: translateY(20px);
  }
  to {
    opacity: 1;
    transform: translateY(0);
  }
}

/* Appliquer l'animation */
.carte {
  animation: apparition 0.6s ease both;
}

/* Décalage pour effet cascade */
.carte:nth-child(2) { animation-delay: 0.1s; }
.carte:nth-child(3) { animation-delay: 0.2s; }
.carte:nth-child(4) { animation-delay: 0.3s; }
Démo interactive — passe ta souris sur le bouton
